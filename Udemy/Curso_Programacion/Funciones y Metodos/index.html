Introducción Funciones y Métodos

Funciones

Una función es un bloque de código que se ejecuta cuando se llama a esa función. Puede pasar datos, conocidos como parámetros, a una función. Una función puede devolver datos como resultado.

Parámetros

Los parámetros son valores que se pasan a una función cuando se llama a la función. Los parámetros se utilizan como variables dentro de la función.

Parámetro REST y SPREAD

El parámetro rest permite a una función aceptar un número indefinido de argumentos como un array.

El parámetro spread permite a una función aceptar argumentos individuales.

Funciones Anónimas y CallBacks

Una función anónima es una función que no tiene un nombre. Se puede utilizar como un valor.

Un callback es una función que se pasa a otra función como un argumento, que luego se invoca dentro de la función externa para completar algún tipo de rutina o acción.

Funciones Flecha

Las funciones flecha, o arrow functions, son una forma compacta y moderna de definir funciones en JavaScript introducida en ES6 (ECMAScript 2015). Se caracterizan por su sintaxis más concisa y por no vincular su propio , lo que las hace útiles en ciertos contextos, como en funciones de callbacks.

Metodos de Busqueda

En JavaScript, los métodos de búsqueda son funciones que se utilizan para localizar elementos en estructuras de datos como cadenas de texto o arreglos. Aquí te explico algunos de los más comunes

Funciones Busqueda para Array

indexOf(): Encuentra el índice del primer elemento que coincide con el valor dado. Si no lo encuentra, devuelve -1.

lastIndexOf(): Igual que indexOf(), pero empieza a buscar desde el final del array.

find(): Devuelve el primer elemento que cumple con una condición especificada en un callback.

findIndex(): Similar a find(), pero devuelve el índice del primer elemento que cumple la condición.

includes(): Verifica si un valor existe en el array. Devuelve true o false.

some(): Comprueba si al menos un elemento del array cumple una condición.

every(): Verifica si todos los elementos cumplen una condición.

Funciones Busquesta para String

indexOf(): Busca un substring y devuelve el índice de su primera aparición.

lastIndexOf(): Busca un substring desde el final del string.

includes(): Determina si el string contiene un substring. Devuelve true o false.

startsWith(): Verifica si el string comienza con un substring.

endsWith(): Verifica si el string termina con un substring.

match(): Busca coincidencias con una expresión regular y devuelve un array con los resultados.

search(): Encuentra el índice de la primera coincidencia con una expresión regular.

Array

Un array es una estructura de datos que permite almacenar múltiples valores en una sola variable. Piensa en un array como una lista ordenada donde cada elemento tiene una posición específica, llamada índice. Estos índices comienzan en 0, lo que significa que el primer elemento tiene el índice 0, el segundo elemento tiene el índice 1, y así sucesivamente.
